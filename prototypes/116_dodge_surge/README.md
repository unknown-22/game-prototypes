# DODGE SURGE

色合わせドッジボールアリーナ。

## 体験仮説

「同じ色のボールを連続で敵に命中させてCOMBOを積み、SUPER THROWで3人まとめて吹き飛ばす瞬間が面白い」

## 実装したコアループ

- プレイヤーは上下キーで投げるボールの色を切り替え、SPACEで投げる
- 敵と同じ色のボールを当てるとCOMBOが伸び、スコア倍率が上昇
- 違う色を当てるとCOMBOリセット＋HEAT上昇
- COMBOが4に達するとSUPER MODE発動（5秒間、全色一致扱い、3倍スコア）
- 敵もボールを投げ返してくる。被弾でHP-1＋HEAT+1
- HEATが10、HPが0、または60秒経過でゲームオーバー

## うまくいっている点

- COMBO積み上げの緊張感とSUPER MODE発動の爽快感のコントラスト
- HEATシステムによるリスクとリターンの明確化（同色を狙い続けるか安全策を取るか）
- 敵の色を見極めて素早く自分の色を切り替える操作の忙しさと達成感

## 次に改善するなら最初に触る点

- 敵AIの多様化（色を切り替える敵、高速移動する敵など）
- パワーアップアイテム（速度アップ、HEATリセットなど）
- BGM・SEの追加
