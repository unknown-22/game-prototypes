# WAKE CHAIN — ウェイクボード連鎖プロトタイプ

## 体験仮説
同じ色のブイを連続でかわしながら通り抜けてコンボを積み上げ、SUPER WAKEで一気に大量のブイを自動回収してスコアが爆発的に伸びる瞬間が面白い。

## 実装したコアループ
- 矢印キー(↑↓)でプレイヤーを上下に移動
- 右から流れてくる4色のブイを通り抜ける
- 同色ブイの連続通過 → COMBOチェイン構築（スコア = 10 × combo）
- COMBO >= 4 → SUPER WAKE発動（5秒間レインボーモード、全色自動回収、スコア3倍）
- 色違いブイ通過 → COMBOリセット + HEAT +1
- HEAT >= 5 または 60秒タイマー切れ → ゲームオーバー
- 航跡パーティクル + CA伝播ビジュアル

## うまくいっている点
- シンプルな操作で直感的に遊べる
- COMBO→SUPER WAKEの流れが気持ちいい
- 60秒のタイマーで短いプレイループ
- HEATシステムでリスク管理の判断がある

## 次に改善するなら最初に触る点
- ブイの速度/パターンのバリエーション追加
- 障害物（岩、他のボート）の導入
- ウェイクジャンプ（SPACEキーでジャンプし空中ブイ回収でボーナス）
