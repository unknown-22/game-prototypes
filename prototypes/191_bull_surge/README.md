# BULL SURGE

Rodeo bull riding color-match COMBO chain game.

## 体験仮説 (Experience Hypothesis)
タイミングよく牛のバック色に合わせる緊張と、連続成功でSUPER RIDEが炸裂する爽快感のコントラストが、短いプレイの中に「もっと上手くなりたい」という再挑戦欲求を生む。

## 実装したコアループ
- 牛が4色（赤/緑/紺/黄）にバック → プレイヤーは対応キーを制限時間内に押してマッチ
- 同色連続マッチでCOMBO構築 → COMBO>=4でSUPER RIDE発動（虹色、スコア3倍、自動マッチ）
- 誤キー/タイムアウトでHEAT蓄積 → HEAT>=100で振り落とされゲームオーバー
- 60秒の時間制限、時間経過でバック間隔短縮（難度上昇）

## うまくいっている点
- 初のロデオジャンル、コレクション内で完全にユニーク
- シンプルな4キー入力で判断の緊張感が生まれる
- SUPER RIDEの視覚的フィードバック（虹色ボーダー、画面揺れ、パーティクル）が気持ちいい
- 55のヘッドレステストが全パス

## 次に改善するなら最初に触る点
- バックの種類を増やす（小バック、大バック、連続バック）
- 牛のアニメーションをより滑らかに
- コンボミュージック/SFXの追加

## 操作方法
- 1/Q/Z: 赤にマッチ
- 2/W/X: 緑にマッチ
- 3/E/C: 紺にマッチ
- 4/R/V: 黄にマッチ
- ENTER: ゲーム開始/再挑戦

## 実行方法
```bash
cd ~/repos/game-prototypes
uv run python prototypes/191_bull_surge/main.py
```
