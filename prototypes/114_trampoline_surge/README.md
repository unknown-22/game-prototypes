# 114_trampoline_surge — TRAMPOLINE SURGE

## 体験仮説
「同じ色のゾーンに着地し続ける緊張感と、空中で回転を決めて大スコアを叩き出す爽快感」

## コアループ
- 左右キーで空中移動し、トランポリンの色付きゾーンに着地
- 同色ゾーン連続着地 → COMBO 増加
- COMBO≥4 → SUPER BOUNCE（虹色、3倍スコア、全色自動一致、5秒間）
- 空中で SPACE → フリップ回転（着地時ボーナス加算）
- 異色着地 → COMBO リセット + HEAT 蓄積
- HEAT≥MAX または タイマー0 → ゲームオーバー

## 操作方法
| キー | 操作 |
|------|------|
| ← → | 空中での左右移動 |
| SPACE | 空中フリップ / タイトル・ゲームオーバーで開始 |
| RETURN | タイトル・ゲームオーバーで開始 |

## リスクとリターン
- **リスク**: 異色ゾーンに着地すると COMBO=0 + HEAT+20
- **リターン**: COMBO 継続でスコア倍率上昇 (combo×10)、SUPER BOUNCE 発動
