# CHROMA SKATE (195)

**Top-down figure skating with color-matching combos on ice.**

## Source
- Generated by game_idea_factory at Score 32.2
- Reinterpreted from: ダイス/バッグ構築ローグライト / 錬金術（合成） #1
- Hooks used: ログ/リプレイが資産 → ice trail echoes, UI連鎖演出 → COMBO chain + SUPER SPIN

## 体験仮説
「氷の上を滑りながら、同じ色のリングを次々に通過してコンボを伸ばし、SUPER SPINで虹色の軌跡を描きながら一気に大量得点を稼ぐ瞬間が面白い」

## Engine
- Pyxel 2.x, 320×240, display_scale=2

## Gameplay
- Ice rink with 4-color rings (RED, GREEN, LIGHT_BLUE, YELLOW)
- Arrow keys/WASD to skate with momentum physics (slippery friction)
- Match ring color to your trail color → COMBO++
- COMBO ≥ 5 → SUPER SPIN: rainbow trail, 3x score, matches all colors (5 seconds)
- Wrong-color ring → COMBO reset, -50 score penalty
- Wall collision → stun + COMBO reset
- 60-second timer, maximize score before time runs out

## Controls
| Key | Action |
|-----|--------|
| ← ↑ ↓ → / WASD | Skate direction |
| ENTER / SPACE | Start / Restart |
