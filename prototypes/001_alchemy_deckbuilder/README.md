# 001_alchemy_deckbuilder — 錬金術デッキ構築ローグライト (Prototype)

> **元アイデア**: Game Idea Factory 生成 (Score 31.85)
> **フック**: 1ターンに使えるカードは「1色だけ」＋ 同色カードが錬金合成で強化
> **エンジン**: [Pyxel](https://github.com/kitao/pyxel) 2.9.5

## 概要

ターン制のデッキ構築バトルプロトタイプ。プレイヤーは手札からカードを選んで敵と戦います。

**核となるルール**: 1ターンにプレイできるカードは**1色（属性）だけ**。  
どの色で攻めるかの判断が勝負の鍵。同色を複数プレイすると**錬金合成**が発動し、効果が増幅されます。

## カードシステム

### 5属性

| 色 | 属性 | 得意 | 合成ボーナス(3枚以上) |
|---|---|---|---|
| 🔴 Red | Fire | 攻撃 | 追加ダメージ |
| 🔵 Blue | Water | 防御・回復 | 追加ブロック |
| 🟢 Green | Earth | マナ生成 | マナ還元 |
| 🟡 Yellow | Air | ドロー・速度 | 追加ドロー |
| 🟣 Purple | Aether | 万能 | 全ボーナス |

### 合成ティア

| 同色枚数 | 名称 | 倍率 |
|---|---|---|
| 2 | DISTILL ★★ | ×1.5 |
| 3 | COMPOUND ★★★ | ×2.0 + 属性ボーナス |
| 4 | ELIXIR ★★★★ | ×3.0 + 属性ボーナス |
| 5+ | MAGNUM OPUS ★★★★★ | ×5.0 + 属性ボーナス |

## 遊び方

1. **起動**: `uv run python prototypes/001_alchemy_deckbuilder/main.py`
2. 手札からカードを**クリック**してプレイ
3. 1色しかプレイできない — どの色で攻めるか戦略的に決める
4. 同じ色を重ねると合成発生！ダメージが跳ね上がる
5. 「End Turn」ボタンでターン終了、敵の反撃を受ける
6. 敵を倒せば勝利、HPが0になると敗北

## 操作

- **マウスクリック**: カード選択 / ボタン操作
- **End Turn**: 右下ボタン
