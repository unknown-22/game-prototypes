# 078 — PARRY CHAIN

Color-match swordplay/fencing game. Enemies dash in from 4 directions with color-coded rings. Press the matching arrow key to parry. Consecutive same-color parries build COMBO → SUPER RIPOSTE.

## Gameplay

### Core Mechanic
1. Enemies spawn at screen edges with color rings: RED(←), LIGHT_BLUE(→), GREEN(↑), YELLOW(↓)
2. Enemies dash toward center
3. Press matching arrow key when enemy is within parry window (40px from center)
4. MATCH → PARRY: combo++, score += 100 × combo
5. MISS → combo reset
6. Enemy reaches center → HP −1
7. COMBO ≥ 5 → SUPER RIPOSTE: auto-destroys all enemies, +500 bonus each

### Risk/Reward
- Match color for COMBO chain but risk wrong guard
- Ghost echo foreshadows next enemy direction

## Controls
- **Arrow keys**: Guard/Parry direction (match enemy color)
- **SPACE**: Start / Restart

## HUD
- Top-left: Score
- Top-center: Combo counter + max combo
- Top-right: HP (♥ × 5)
- Bottom-center: Timer (90s)
