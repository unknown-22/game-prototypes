# TIGHTROPE SURGE — 105

Color-match tightrope walking game. Balance on a high-wire, collecting same-color gems to build COMBO chains. Reach COMBO ≥ 4 to activate SUPER BALANCE mode.

## Source
Generated by [game_idea_factory](https://github.com/unknown-22/game_idea_factory), 2026-06-07.
All 10 ideas were deckbuilder/auto-shooter. Reinterpreted Idea #1 (Score 32.35, hooks: "synthesis compression" + "circuit/pipe visualization") into tightrope walking — an untapped genre.

## Engine
- Pyxel 2.x, 320×240, display_scale=2
- Python 3.12+, single file (~640 lines)

## Gameplay
1. Walk forward automatically on the rope
2. LEFT/RIGHT arrows to balance (stay centered)
3. Step on gems — same color = COMBO↑, wrong color = COMBO reset + HEAT↑
4. COMBO ≥ 4 → SUPER BALANCE (5s: wind immunity, 3x score, rainbow trail)
5. HEAT reaches 100% → fall off rope → GAME OVER
6. Wind gusts push you left/right — counter-balance!

## Controls
- LEFT ARROW: lean left
- RIGHT ARROW: lean right
- RETURN: start / restart

## Risk & Reward
- Risk: chasing same-color gems near the rope edge increases fall risk
- Reward: COMBO chain → SUPER BALANCE → huge score multiplier
- Safety: ignore color matching, stay centered → 1x score only

## "Most Fun Moment" (面白い瞬間)
同じ色の宝石を連続で踏んでCOMBOを溜め、SUPER BALANCEで風もブレも無効化して高速で突き進む瞬間
