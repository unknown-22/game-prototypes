## Gameplay

1. **PLACE**: Click cells on a 16×12 grid to place colored dominoes (4 colors cycle with keys 1-4)
2. **TRIGGER**: Press SPACE to start the chain reaction from the top-center start cell
3. **ANIMATE**: Watch the BFS chain propagation
   - Same-color consecutive topples build **COMBO** multiplier
   - Cells with 2+ outgoing neighbors create **SPLIT** paths (+200 pts)
   - When two propagation paths reach the same cell → **CONVERGE SUPER BURST** (+1000 pts, explosion particles)
4. **SCORE**: Chain length × 10 + combo steps × 50 + converge/split bonuses
5. **RETRY**: SPACE to rebuild a new layout

## Controls

| Key | Action |
|-----|--------|
| Mouse click | Place domino at hovered cell |
| 1/2/3/4 | Select color (RED/GREEN/BLUE/YELLOW) |
| SPACE | Advance phase / trigger chain |
| ESC | Quit to title |
