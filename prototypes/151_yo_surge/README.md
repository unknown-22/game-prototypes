# YO SURGE — Yo-Yo Skill Toy Game

**Prototype #151** | 2026-06-23

## 体験仮説

「同じ色の宝石を連続でキャッチしてCOMBOを積み上げ、SUPER YO-YOが発動した瞬間に『操作を極めた』感覚を得る」

## 実装したコアループ

マウスでヨーヨーを操作し、4色（赤・緑・水色・黄）の落下する宝石をキャッチする。
- 同色連続キャッチでCOMBO+1、スコアに倍率がかかる
- COMBO≥4でSUPER YO-YO発動（5秒間、全色キャッチ可能、スコア3倍、回収範囲2倍、HEAT無効化）
- 色違いキャッチでCOMBOリセット＋HEAT+0.5、取り逃しでHEAT+1
- HEAT≥15 または タイマー0でゲームオーバー

## Source
ゲームアイデアファクトリー #1（Score 31.45 — ダイス/バッグ構築ローグライト/ハッキング）
- フック「ログ/リプレイが資産」→ 前回キャッチした色が次のアクティブ色に
- フック「1ターンに使えるカードは1色だけ」→ 同時にキャッチできる色は1色のみ

## うまくいっている点

- ヨーヨーのlerp追従が物理的で気持ちいい操作感を生む
- COMBO→SUPERの流れが明確で、爆発的なスコアの伸びを実感できる
- HEATバーが視覚的なリスク指標として機能
- タイマー＋難度上昇で緊迫感が増す
- 初のヨーヨー/スキルトイジャンル

## 次に改善するなら最初に触る点

- ヨーヨーに「投げる」動作やトリックの要素を追加
- 宝石の色にも特殊効果（スピード変化、スコア倍率、追加時間）を持たせる
- ワイヤー/ストリングの長さ変化による戦略性

## 操作方法

- マウス移動：ヨーヨー操作
- クリック/SPACE：タイトル→開始 / ゲームオーバー→再挑戦
