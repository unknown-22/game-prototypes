# WRECK CHAIN (147)

## 体験仮説
「同じ色のブロックを狙い続ける集中と、SUPER BALL発動後の爽快な無双状態のコントラストが面白い」

## コアループ
1. マウスでクレーンアンカーを左右に操作
2. クリックで振り子鉄球を解放
3. 鉄球がビルのブロックを破壊（同色連続でCOMBOチェイン構築）
4. ブロック破壊で破片が飛び散り、上のブロックが重力崩落（CHAINボーナス）
5. COMBO>=4でSUPER BALL発動（5秒間レインボー、全色マッチ＋スコア3倍）
6. 色違い破壊でHEAT蓄積（>=100でゲームオーバー）
7. 60秒以内に全ブロック破壊で勝利

## うまくいっている点
- 振り子物理による「振り抜き」感覚が気持ちいい
- COMBO構築→SUPER BALLの報酬ループが明確
- HEATリスクで無闇な破壊にペナルティ
- 重力崩落CHAINが予期せぬスコア爆発を生む

## 次に改善するなら最初に触る点
- ブロックの色分布をプレイヤーが把握しやすくする（現在はランダム）
- 複数種類のビル形状（L字型、ピラミッド型など）
- 音（pyxel.playを活用したSE）

## 操作方法
- マウス移動: クレーンアンカー位置
- マウスクリック（離す）: 鉄球を振り子解放
- SPACE: タイトル画面から開始
- R: ゲームオーバー/勝利から再挑戦

## 実行方法
```bash
cd ~/repos/game-prototypes
uv run python prototypes/147_wreck_chain/main.py
```
