# CLAW SURGE — 187

クレーンゲーム × 色合わせCOMBOチェイン。
コレクション初のクレーンゲーム/アーケードジャンル。

## 体験仮説

「同色の景品を連続で掴むか、リスクを取って異色で点数を稼ぐかの選択が毎回発生し、SUPER CLAW発動時の気持ちよさが再挑戦を促す」

## 遊びの核（面白い瞬間）

「同色の景品を4連続で掴んでSUPER CLAWが炸裂し、虹色のアームが全ての景品を吸い寄せて掴みまくり、スコアが3倍に跳ね上がる瞬間」

## 元ネタ

game_idea_factory Idea #1 (Score 32.2) — 発電所（過負荷と放電）ダイス/バッグ構築ローグライト
- 「ログ/リプレイが資産」→ ゴーストクローのベストラン軌跡表示
- 「UIの連鎖演出（崩壊/増加/圧縮）」→ COMBO連鎖 → SUPER CLAW圧縮

## 操作方法

| 操作 | キー |
|------|------|
| クロー移動 | 矢印キー / WASD |
| 掴む | SPACE / マウスクリック |
| ゲーム開始/リトライ | SPACE / ENTER / クリック |

## ルール

- 4×5グリッドに色付き景品（赤/緑/紺/黄）が配置される
- 各景品にはNORMAL（10点）とRARE（30点、光る）の2段階
- 同色連続掴みでCOMBOが伸び、スコア倍率が上昇（1.0 → 1.5 → 2.0 → 2.5 → ... 最大5.0）
- COMBOが4以上でSUPER CLAW発動（300フレーム、虹色、3倍スコア、全色自動マッチ）
- 異色掴みでCOMBOリセット + HEAT +15
- HEAT >= 100 または タイマー0 でゲームオーバー
- HEATは自動減衰（0.05/フレーム）
- 景品は掴まれてから90フレーム後にリスポーン（色・レア度がランダム変化）
- ゴースト軌跡が前回ベストランのクロー軌跡を表示

## リスクとリターン

- **同色連続**: COMBO倍率上昇、SUPER CLAWへの道 → 高報酬
- **異色掴み**: COMBOリセット、HEAT増加 → ゲームオーバーリスク
- **SUPER CLAW中**: 全色安全、3倍スコア → 短期決戦の大チャンス
