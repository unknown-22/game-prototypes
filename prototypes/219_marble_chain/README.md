# 219 Marble Chain

マーブルラン（ビー玉レース）プロトタイプ。同色ビー玉をスイッチで経路誘導し、COMBO連鎖でスコア倍増。

## 体験仮説
「スイッチを切り替えて同色ビー玉を同じレーンに集め、COMBO連鎖でスコアが倍増する瞬間が最高に気持ちいい」

## 実装したコアループ
1. 4色（赤/緑/青/黄）のビー玉が上部中央から一定間隔で出現
2. 3レーン（左/中央/右）を落下
3. 2つのスイッチ（y=80, y=150）をクリックで切り替え、ビー玉の経路を変更
4. 下部で回収時、同色ならCOMBO継続（スコア倍率上昇）、異色ならHEAT上昇+COMBOリセット
5. COMBO≥4でSUPER MODE発動（5秒間全色マッチ、3倍スコア）
6. 60秒タイマー切れ または HEAT≥100 でゲームオーバー
