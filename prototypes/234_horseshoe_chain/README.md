# HORSESHOE CHAIN (234_horseshoe_chain)

色合わせホースシュー（蹄鉄）投げ COMBO チェインゲーム。

## ソース
- game_idea_factory #1 (Score 31.8): ハッキング/回路デッキ構築ローグライト
  - フック: 「ログ/リプレイが資産」→ ゴースト軌跡、「落下/重力で連鎖崩壊」→ 放物線 COMBO 連鎖
  - 未開拓ジャンル「Horseshoes」に再解釈

## エンジン
- Pyxel 2.x, 320×240

## 体験仮説
「放物線の軌道を読み切り、同色の杭に連続で蹄鉄をリングインさせて COMBO を積み重ね、SUPER TOSS で爆発的スコアを得る爽快感」

## メカニクス仮説
- **Mechanics**: マウスドラッグによる放物線照準＋重力物理、同色連続リンガーで COMBO 加算、COMBO≥4 で SUPER TOSS 発動
- **Dynamics**: 同色杭を狙う高リスク高リターン vs 安全に外さない投げ方のジレンマ
- **Aesthetics**: 正確な狙いが決まった時の達成感、SUPER TOSS の爆発的カタルシス

## ゲームプレイ
- **コアメカニクス**: マウスクリック＆ドラッグで放物線を描いて蹄鉄を投げる。画面下部の4色杭（赤/黄緑/紺/黄）に同色蹄鉄を連続でリングインさせると COMBO が伸びる
- **SUPER TOSS**: COMBO≥4 で発動（5秒間虹色、全色マッチ、スコア3倍）
- **HEAT リスク**: 色違いリンガー +15、空振り +10、自然減衰 0.02/f、上限 100 でゲームオーバー
- **難度上昇**: 15秒毎に杭の色がシャッフル、残り10秒で杭点滅＋HEAT 減衰停止（最終局面）
- **ゴースト軌跡**: ベストスローの弾道を半透明で表示

## 操作
- マウスのみ
- クリック＆ドラッグ: 狙いとパワーを設定（遠くにドラッグするほど強く投げる）
- リリース: 蹄鉄を投げる
- ゲームオーバー画面でクリック: リトライ
