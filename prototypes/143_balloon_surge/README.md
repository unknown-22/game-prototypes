# BALLOON SURGE (143)

熱気球が色付きリングを通過しながら上昇する、色合わせCOMBO連鎖ゲーム。

## 体験仮説
「熱が限界近くで燃料が尽きかけている緊張の中、同色リングを連続でくぐり抜けてSUPER LIFTを発動し、虹色バルーンが大量のリングを自動回収してスコアが爆発する瞬間に、リスクを取った甲斐があったと感じる」

## コアループ
1. 画面中央に固定された気球を左右キーで操舵し、下から迫る色付きリングを通過する
2. 同色リングを連続通過 → COMBO上昇（+COMBO×5のスコアボーナス）
3. COMBO≥5 → SUPER LIFT発動（5秒間虹色、全リング自動同色判定、スコア3倍、燃料消費なし）
4. 色違い通過 → COMBOリセット + 熱15増加
5. SPACEキーで燃料を燃やして高速上昇（スコア効率UP）するが熱が蓄積
6. 熱≥100 → GAME OVER / 燃料切れでも操舵は可能

## エンジン
- Pyxel 2.x, 320×240, display_scale=2
- Python 3.12+

## 操作方法
- LEFT/RIGHT (←→) — 気球を左右に操舵
- SPACE — 燃料を燃やして高速上昇（熱が蓄積）
- ENTER (RETURN) — タイトル画面で開始 / ゲームオーバーで再挑戦

## リソース
- **燃料 (Fuel)**: 0-100。SPACEで消費（-0.3/frame）、自然減少（-0.05/frame）
- **熱 (Heat)**: 0-100。燃焼で+0.15/frame、色違い通過で+15、自然冷却-0.05/frame。100でゲームオーバー
- **COMBO**: 同色連続通過で+1、色違いで1にリセット。5以上でSUPER LIFT
