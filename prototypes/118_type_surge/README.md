# TYPE SURGE — 色合わせタイピング防衛ゲーム

**Prototype #118** | 2026-06-10

## 体験仮説

「同じ色の文字を連続で打ち抜いてCOMBOを積み上げ、SUPER TYPERで画面上の文字を一気に消し去る爆発的爽快感」
→ 色を認識→指を動かす→結果が即座にフィードバックされる高速ループが、リズムゲームに近い没入感を生む。

## 遊びの核

- 上空から色付きアルファベットが降ってくる
- 同じ文字のキーをタイプして破壊
- **同じ色**を連続ヒット → COMBO上昇
- COMBO x4 → **SUPER TYPER**（5秒間、全文字を自動破壊 + 3倍スコア）
- 違う色でヒット → COMBOリセット + HEAT
- ミスタイプ → HEAT +15
- 文字を逃す → HP -1 + HEAT +5
- HEAT 100 → キーボードロック（スタン）

## リスクとリターン

- **同色狙い**: COMBOで高倍率だが、違う色の文字が溜まっていく危険
- **安全プレイ**: 色を気にせず落ちてくる順に打つ。COMBOは伸びないがHEATは溜まらない
- **SUPER TYPER**: 画面が詰まった時の緊急回避にも、計画的にCOMBOを貯めて発動する爆弾にも

## 操作方法

- **A-Z キー**: 落下する文字をタイプして破壊
- **SPACE**: ゲーム開始 / リスタート

## 仕様

- エンジン: Pyxel 2.x, 320×240, 60fps
- 4色モード (RED, GREEN, DARK_BLUE, YELLOW) → 60秒以降 5色 (+ORANGE)
- 90秒の制限時間
- 落下速度・出現頻度が徐々に上昇
- 30秒以降: ペア出現あり

## スコア計算

- 通常ヒット: 10 + COMBO × 5
- SUPER TYPER中: 30 + COMBO × 15 (実質3倍)

## 画面構成

- **タイトル画面**: ゲームタイトル、操作説明
- **ゲーム画面**: 落下文字、HUD（スコア/COMBO/HP/タイマー/HEATバー）、パーティクル/フローティングテキスト
- **ゲームオーバー画面**: 最終スコア、最大コンボ、破壊文字数
