# 084_bar_chain — BAR CHAIN (バーチェーン)

バーテンディング & ドリンクミキシングプロトタイプ。
Pyxel 2.x / 320×240 / ジャンル: バーテンディング（コレクション初）

## Source

- 発案: game_idea_factory #1 (Score 32.75) — 発電所（過負荷と放電）デッキ構築
- 再解釈: デッキ構築フックを未開拓バーテンディングジャンルに変換
  - 「合成圧縮」→ 同色連続注ぎ = COMBOチェーン
  - 「CAグリッド埋め」→ オーダーキュー緊急度
  - heat/risk → 誤注ぎHEAT / コンボ延長リスク

## 体験仮説

> ギリギリまで同色COMBOチェーンを伸ばしてからSERVEし、大量スコアを獲得するのが面白い

## メカニクス

- 4色リキッド: RED / GREEN / BLUE / YELLOW
- オーダー: 色の並びが指定される（例: RED, RED, GREEN）
- 同色連続注ぎ → COMBO倍率上昇（×1, ×2, ×3... 上限×6）
- 色違い注ぎ → COMBOリセット + オーダー不適合時HEAT+1
- 全ターゲット色注ぎ完了 → SERVEボタン有効化
- 余剰注ぎ → コンボ倍率ボーナスで追加得点
- HEAT最大(10) → ゲームオーバー
- SERVE成功 → HEAT-2回復

## 操作

| 操作 | 内容 |
|------|------|
| マウスクリック（ボトル） | リキッドを注ぐ |
| マウスクリック（SERVEボタン） | ドリンク提供 |
| Rキー（GAME OVER画面） | リスタート |

## 面白い瞬間

同色ボトルを連打して COMBO ×5 を達成し、オーダーを満たして SERVE ボタンを押すと大量スコアが表示される瞬間。

## リスクとリターン

- 同色COMBO継続で高倍率スコア（最大×6+）が得られる
- 色違い注ぎでHEAT蓄積 → HEAT=10でゲームオーバー
- 「もう1回同じ色を注いでコンボを伸ばすか、安全にSERVEするか」の判断
