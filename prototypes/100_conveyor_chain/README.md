## 体験仮説
「同じ色のアイテムがコンベア上で連鎖し、合成されて爆発的に得点が伸びる」のを見て操作するのが面白い

## エンジン
- Pyxel 2.x, 320×240, display_scale=2
- 開発: OpenCode CLI (opencode-go/deepseek-v4-pro, 一発成功)

## ゲームプレイ

### コアメカニクス
- 8×8グリッド上のコンベアベルトタイルをクリックで90°回転
- 上部から色付きアイテムが出現し、ベルトに沿って流れる
- **同じ色のアイテムが隣接 → チェインリンク形成**
- **チェイン3以上 → COMBO発動、先頭アイテムがSUPERに（レインボー、3倍スコア）**
- アイテムが下端の出力ビンに到達 → 色一致でスコア、不一致でHEAT
- HEAT 100 = GAME OVER

### リスクとリターン
- **コンボを狙う** → 同じ色のレーンに集中させる（高得点だが経路制約）
- **SUPERアイテム** → どのビンでも受け入れ（安全だが出現はCOMBO依存）
- **HEAT管理** → 間違った経路はHEATが溜まる

## 操作
| 操作 | 内容 |
|------|------|
| マウスクリック | グリッドセルを時計回りに90°回転 |
| SPACE / ENTER | タイトル画面で開始 |
| R | ゲームオーバー画面でリスタート |
