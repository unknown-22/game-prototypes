# STACK SURGE — Prototype #080

ジェンガ風タワー引き抜きゲーム。4色のブロックで構成されたタワーからブロックを引き抜き、同色連続でCOMBOを構築する。

## 体験仮説
「同色ブロックを連続で引き抜く緊張感と、COMBOが3に達してSYNTHESISが発動し不安定ブロックが一気に安定化する解放感」

## 元アイデア
game_idea_factory #1 (Score 32.35) — ハッキング（回路/ログ）ヴァンサバ亜種
- フック「効果が合成されて1枚に圧縮」→ COMBO → SYNTHESIS
- フック「CAで盤面が感染/増殖」→ 不安定化の隣接伝播

## コアループ
1. タワー（8段・各段3ブロック）からブロックをクリックで引き抜く
2. 前回と同じ色なら COMBO+1、違う色なら COMBO=0 + 隣接ブロック不安定化（CA伝播）
3. COMBO ≥ 3 で SYNTHESIS: 全不安定ブロックが安定化 + COMBO×100 ボーナス
4. 引き抜いたブロックはタワー上部に新しい段として追加
5. 不安定率 > 50% でタワー崩壊（GAME OVER）

## リスクとリターン
- 同色COMBO継続 → 高倍率スコアだが、別色を避けるため危険な位置のブロックも狙う必要あり
- 安全に別色を引く → COMBOリセット + 不安定化が広がる（CA伝播）
- 不安定ブロックを引く → HEAT上昇（5でゲームオーバー）
- SYNTHESISで不安定化を一掃できるが、COMBO 3に達するまでのリスクを取る必要がある

## 操作
- マウスクリック: ブロック引き抜き
- SPACE / ENTER: タイトルから開始 / ゲームオーバーから再挑戦
