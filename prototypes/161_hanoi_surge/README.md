## Gameplay

哲学仮説: 「同色のディスクを連続で動かしてCOMBOチェインを構築し、SUPER MOVEで一気に形勢逆転するのが面白い」

1. **SELECT**: Click a peg to select the top disk
2. **MOVE**: Click another peg to move the disk (only smaller on larger allowed)
3. **COMBO**: Same-color consecutive moves build a COMBO multiplier
4. **SUPER MOVE**: COMBO≥4 triggers SUPER MOVE — one free move ignoring size rules
5. **GOAL**: Stack all 5 disks on the rightmost peg in correct size order (largest at bottom)
6. **RISK**: Invalid moves (large on small) generate HEAT; HEAT≥100 = game over
7. **TIME**: 120-second countdown; time-out = game over

## Most Fun Moment

同色ディスクのCOMBOチェインが4に達し、SUPER MOVEが発動。サイズ制限を無視した大胆な一手で、一気にゴールに近づく形勢逆転の気持ちよさ。

## Risk & Reward

- **COMBO狙い**: 同色の移動順序を計画すればCOMBOが伸びSUPER MOVEを獲得できるが、最短手順から外れるリスクがある
- **SUPER MOVEの温存**: 強力だがチャージを消費。難所で使うか、早期に使ってCOMBOを再構築するかの判断
- **HEATプレッシャー**: 無効な移動はHEATを溜め、最大100でゲームオーバー。強引な移動戦略か慎重な手順かのバランス
